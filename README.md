<h1>HOW to run</h1>

1. Node.js, npm 설치

2. 프로젝트 clone
(git bash 창에서)
<code>
   git clone https://github.com/flying-adventure/rockscissiorspaper.git
   cd RockScissiosPaper_TeachableMachine
   npm install
   npm run start
</code>
<h2>And go to http://localhost:8080</h2>
